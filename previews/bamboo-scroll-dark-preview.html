夜间冷墨蓝调

护眼基底

从纯灰调整为冷墨蓝 #151921，降低暖黄光刺激，夜间观看更深邃舒适。

结构七色

分离关键词（赭色）与类型（缃色）、属性（黛蓝）与常数（胭脂），避免大段同色。

对比度提升

注释文字采用银灰 #8E8578，对比度达到 WCAG AA 4.6:1 级别，弱光下不眯眼。

支持全面

新增括号彩虹色、补全搜索与高亮UI、定制了 Python 和 Java 乃至 Git 差异面板。
